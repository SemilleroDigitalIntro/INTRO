"¿Por qué elegir la lactancia materna?""

Beneficios para la madre

  1. Favorece la Recuperación Postparto:

    2. Estimula la contracción del útero, ayudando a reducir el sangrado.

  2. Ayuda a Perder Peso:

    3. Quema calorías adicionales mientras produces leche.

  3. Reduce el Riesgo de Enfermedades:

    4. Protege contra cáncer de mama, ovarios y osteoporosis.

  4. Conveniente y Económica:

    5. Siempre está disponible, a la temperatura adecuada, y no requiere preparación.

  5. Fortalece el Vínculo:

    6. Promueve momentos únicos de cercanía con el bebé.

Beneficios para el bebé

  1. Nutrición Completa:

    2. Contiene todos los nutrientes que el bebé necesita para crecer sano.

  2. Fortalece el Sistema Inmunológico:

    3. Protege contra infecciones, alergias y enfermedades como diarrea y neumonía.

  3. Mejora el Desarrollo Cerebral:

    4. Favorece el desarrollo cognitivo gracias a sus ácidos grasos esenciales.

  4. Vínculo Emocional:

    5. Fomenta una conexión profunda con la madre.

  5. Menor Riesgo en ENfermedades Crónicas:

    6. Disminuye la probabilidad de obesidad, diabetes y asma en el futuro.
